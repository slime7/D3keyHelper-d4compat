# D4KeyHelper

D4KeyHelper 是一个基于 AutoHotkey v2 的暗黑4按键宏工具，提供图形界面和可保存的多配置方案。

运行代码请使用 AutoHotkey v2.0 或更新版本。

## 功能范围

当前版本只保留战斗相关功能：

1. 6 个技能位的按键策略：禁用、按住不放、连点、保持Buff、按键触发。
2. 配置快速切换，以及切换后自动启动宏。
3. 宏启动方式：懒人模式、仅按下时、仅按一次。
4. 走位辅助、药水辅助、快速暂停、单线程按键队列。
5. 智能暂停：按 Tab 暂停宏，按 Enter、T、M 停止宏。
6. 左键技能、右键技能可勾选后改为自定义快捷键；取消勾选后回到原始鼠标键且禁用。

已移除 F5 助手和所有物品相关功能，包括赌博、拾取、分解、重铸、升级、转化、丢装、背包识别、物品颜色识别、安全格设置。

## 配置文件

脚本启动时会读取同目录下的 `d4keyhelper.ini`。如果文件不存在，会在下次关闭或退出时按当前 UI 设置创建。

`General` 区块保留的主要配置：

1. `gameonly`：默认为 1，仅在暗黑4窗口生效；改为 0 后对任意窗口生效。
2. `gameresolution`：默认为 `Auto`，也可手动设置为 `宽x高`。
3. `gamegamma`：像素识别的 gamma 修正值，默认为 1.000000。
4. `buffx_1` 到 `buffx_6`、`buffy_1` 到 `buffy_6`：保持Buff取样点，按 `buffreferencewidth=3440`、`buffreferenceheight=1440` 的参考坐标填写。
5. `buffgreenactive`：取样点绿色通道达到此值时视为 Buff 存在，默认为 95。
6. `runonstart`：默认为 1，宏启动时立即执行一次启用的策略。
7. `sendmode`：按键发送模式，默认为 `Event`。

保持Buff默认取样点按暗黑4默认技能栏位置估算；如果调整了 HUD 位置，可直接改配置文件中的 Buff 坐标。保持药水CD的取样坐标仍可能需要按实际窗口和 UI 显示微调脚本。
